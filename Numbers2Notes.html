Numbers2Notes Requirements

Welcome!

This page shows what is required for Numbers2Notes to work, what is recommended, and which older items are no longer needed.

Start with the Required section first.

New to REAPER customization?
If you do not already have ReaPack and SWS installed, ReaBoot.com is an easy way to get started.

1. Required

These are the items users should install first.

  • ReaTeam Extensions Required
    Must include: SWS, ReaImGui, and js_ReaScriptAPI.
    https://github.com/ReaTeam/Extensions/raw/master/index.xml
  • ReaTeam Scripts Repository Required
    Install through ReaPack.
  • AHS Music Tech (Rock Kennedy) - Numbers2Notes Main Repository Required
    Import this repository into ReaPack to install the main Numbers2Notes scripts and included N2N JSFX.
    https://github.com/rockumk/AHS_Music_Tech/raw/master/index.xml
  • Included with REAPER (Cockos / Stillwell) Required
    Some required effects are already included with REAPER and do not need a separate download.
  • Geraint Luff / Geraint's JSFX Required
    Needed for Pad Synth related functions.
    https://github.com/geraintluff/jsfx-pad-synth
  • Tukan Studios Required
    Install through ReaPack.
    https://raw.githubusercontent.com/TukanStudios/TUKAN_STUDIOS_PLUGINS/main/index2.xml
  • Saike Tools Required
    Install through ReaPack.
    https://raw.githubusercontent.com/JoepVanlier/JSFX/master/index.xml
  • Chmaha's Airwindows Ports Required
    Install through ReaPack.
    https://github.com/chmaha/airwindows-JSFX-ports/raw/main/index.xml
  • Vital Required
    Download Vital →
  • Surge XT Required
    Download Surge XT →
  • Airwindows Required
    Needed effects include: Calibre, Isolator, Holt, Tube, and Drive.
    Download / Visit Airwindows →
  • MONSTER Drums v3 Plugin Files Required
    You need both the plugin files and the library files.

    Google Drive Folder →

    Alternative PC Download →

    Alternative Mac Download →

  • MONSTER Drums v3 Library Files Required

    Google Drive Folder →

    Alternative Library Download →

  • N2N LibreArp Preset Library Required
    Required support file.
    Download N2N_Librearp_Presets.RPL →
  • N2N Surge XT Preset Library Required
    Required support file.
    Download N2N_SurgeXT_Presets.RPL →
