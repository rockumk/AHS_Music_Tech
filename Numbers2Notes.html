Numbers2Notes Requirements

Welcome!

This page shows the repositories and files needed for Numbers2Notes, followed by a few other recommended plugins.

Please begin by adding the repositories below into ReaPack, then install the required files and plugins, import the presets and you are ready to start.

New to REAPER customization?
If you do not already have ReaPack and SWS installed, ReaBoot.com is an easy way to get started.

2. Required Files and Plugins

These are the required items users should install after adding the repositories above.

  • ReaTeam Extensions Components Required
    Must include: SWS, ReaImGui, and js_ReaScriptAPI.
  • AHS Music Tech (Rock Kennedy) - Numbers2Notes Main Repository Required
    Import this repository into ReaPack to install the main Numbers2Notes scripts and included N2N JSFX.
    The link is already included in the list of repositiories above!!!
  • Included with REAPER (Cockos / Stillwell) Required
    Some required effects are already included with REAPER and do not need a separate download.
  • Vital Required
    Download Vital →
  • Surge XT Required
    Download Surge XT →
  • Airwindows Required
    Needed effects include: Calibre, Isolator, Holt, Tube, and Drive.
    Download / Visit Airwindows →
  • MONSTER Drums v3 Plugin Files Required
    You need both the plugin files and the library files.

    Google Drive Folder →

    Alternative PC Download →

    Alternative Mac Download →

  • MONSTER Drums v3 Library Files Required

    Google Drive Folder →

    Alternative Library Download →

  • N2N Preset Libraries Required - Make sure you install the presets for each FX and restart Reaper before starting your first project with Numbers2Notes. You should also run the N2N_Drum_System.lua script at least once after installation and anytime you are using N2N Drum Arranger.
    Download N2N_Librearp_Presets.RPL →

3. Other Recommended Plugins
