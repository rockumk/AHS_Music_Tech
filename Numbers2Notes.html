Numbers2Notes Setup Guide

Welcome!

This page shows what is required, what is recommended, and what is optional for Numbers2Notes.

Install the required items first. After that, add recommended plugins if you want the full intended experience.

Starting from scratch?
If you do not already have ReaPack and SWS installed, ReaBoot is a simple way to get the basics in place.

1. Required for Numbers2Notes to Run

These are the core dependencies. Numbers2Notes may not run correctly without them.

  • ReaTeam Extensions
    Required
    Must include: ReaImGui, SWS, and js_ReaScriptAPI.
    https://github.com/ReaTeam/Extensions/raw/master/index.xml
  • AHS Music Tech (Rock Kennedy) - Main Numbers2Notes Repository
    Required
    Import this repository into ReaPack to install the main Numbers2Notes scripts and included N2N JSFX.
    https://github.com/rockumk/AHS_Music_Tech/raw/master/index.xml
  • Included with REAPER (Cockos / Stillwell)
    Required where referenced
    Some effects used by Numbers2Notes are included with REAPER itself and do not require any separate download.

  • ReaTeam Scripts Repository

6. Preset Libraries

If you are using the supplied Surge XT and LibreArp presets, import these libraries inside the plugin preset browser.

  • N2N LibreArp Preset Library
    Recommended if using LibreArp
    Import the .RPL file from inside LibreArp.
    Download N2N_Librearp_Presets.RPL
  • N2N Surge XT Preset Library
    Recommended if using Surge XT
    Import the .RPL file from inside Surge XT.
    Download N2N_SurgeXT_Presets.RPL

7. Quick Setup Order

  • Install ReaPack, SWS, ReaImGui, and js_ReaScriptAPI.
  • Import the AHS Music Tech repository.
  • Add the recommended ReaPack repositories you want to use.
  • Install recommended plugins such as Surge XT, Vital, STFU, and Airwindows.
  • If using MONSTER Drums v3, install both the plugin files and the library files.
  • Import preset libraries for Surge XT and LibreArp if desired.
Note:
Numbers2Notes can still warn users at startup when some optional or recommended plugins are missing. If needed, that warning behavior can be adjusted in the script configuration.
