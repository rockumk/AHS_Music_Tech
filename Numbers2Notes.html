Numbers2Notes Requirements

Welcome Beta Testers!

Numbers2Notes is a powerful script that relies on specific extensions and plugins to function correctly. Please follow this guide to set up your environment.

New to Reaper Customization?
If you do not have ReaPack or SWS installed yet, the easiest way to get started is to run the default recipe from Reaboot.com. This will install the package manager and core extensions automatically.

1. Essential Extensions (The Engine)

The script will not run without these extensions. Ensure these are installed via ReaPack.

  • ReaTeam Extensions System Core
    Must include: ReaImGui (Interface), SWS, and js_ReaScriptAPI.
    https://github.com/ReaTeam/Extensions/raw/master/index.xml

2. JSFX Repositories (The Effects)

Import these repositories into ReaPack (Extensions > ReaPack > Import Repositories...) to get the required audio processors.

  • Tukan Studios Essential Mixing
    https://raw.githubusercontent.com/TukanStudios/TUKAN_STUDIOS_PLUGINS/main/index2.xml
  • Saike Tools Sequencers
    https://raw.githubusercontent.com/JoepVanlier/JSFX/master/index.xml
  • Geraint's JSFX Pad Synth
    https://geraintluff.github.io/jsfx/index.xml

3. VST Instruments & Effects (Direct Download)

Download and install these free plugins to ensure the default band sounds correct.

  • Surge XT Synthesizer Download →
  • Sitala Drums Download →
  • LibreARP Arpeggiator Download →
  • Drum8 (Audiolatry) Drums Download →
  • Tattoo (Audio Damage) Percussion Download →
  • STFU (Zeek) Sidechain/Gate Download →
  • Airwindows Mixing Tools
    Needed: Calibre, Isolator, Holt, Tube, Drive     Download →

4. Install Numbers2Notes

Once the environment is ready, import this repository to install the main script.

  • AHS Music Tech (Rock Kennedy) Main Script
    https://github.com/rockumk/AHS_Music_Tech/raw/master/index.xml

Note: If you prefer not to install specific VSTs, the script will warn you on startup. You can disable this warning by editing numbers2notes_config.lua inside the script folder.
